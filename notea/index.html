Why Notea

Purpose-built for organisations that cannot compromise on governance.

Most AI tooling asks teams to upload sensitive files into another vendor’s silo and accept generic output. Notea is designed for organisations that need data sovereignty, cryptographic control, and consistent reporting standards.

Security Model

Public marketing stays air-gapped from the clinical application.

This route is intentionally static. There are no server-side actions here, no secrets, and no direct database access. Prospects discover Notea on Achieve Therapy’s GitHub Pages site, then move into the authenticated app only when they are ready to create a workspace.

  • Static site delivery through GitHub Pages and CDN caching
  • Authenticated application and onboarding flow on Railway
  • Stripe subscription created only after verified sign-in
  • No public-facing billing logic on the marketing page
